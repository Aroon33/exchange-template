CryptoX の特徴（UIイメージ）

クライアントへの提案・モックアップとして、そのままデモ可能なレベルの UI です。

1. プロ仕様の取引画面

板・約定履歴・注文パネルを 1つの画面で操作できます。

2. ウォレット & 入出金（JPY対応）

ビットコインだけでなく 日本円 の入出金UIも搭載。

3. システム取引 UI

戦略テンプレート選択・損切り/利確・最大注文数など設定可能。
